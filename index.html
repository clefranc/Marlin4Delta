Welcome to my forked Marlin firmware.

This custom firmware is specifically design to print objects with dimensional accuracy on deltas.

Delta printers have both physical and virutal parameters that affect geometry of printed objects. This fork is meant to properly set these parameters. For more information on configuration and other useful Marlin4Dev functionalities, see the wiki

Marlin was originally derived from Sprinter and Grbl by Erik van der Zalm in 2011. Its main repository is located at GitHub: https://github.com/MarlinFirmware/Marlin and a developer repository: https://github.com/MarlinFirmware/MarlinDev. Delta robot code was introduced in June 2013.

Since 2015, Marlin have migrated to its own GitHub organization: https://github.com/MarlinFirmware.

While we can't prevent the use of this code in products (3D printers, CNC, etc.) that are closed source or crippled by a patent, we would prefer that you choose another firmware or, better yet, make your own.
